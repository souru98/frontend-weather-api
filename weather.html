Weather App

Build a simple UI that shows the weather of a city that is
entered by the user. Use HTML, CSS and JavaScript to
achieve this.

Requirements:

  1. Use open weather map API to achieve this.https://openweathermap.org/
  2. You need to create an account in openweathermap.org
  3. Display the following details
    1. Current date and time
    2. Weather(can be random)
    3. Temperature(conversion from one unit to another)
    4. Text box with autocomplete
